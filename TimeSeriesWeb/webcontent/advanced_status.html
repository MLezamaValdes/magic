On this page you can view the time span of data that each plot contains.

First choose a set of plots and then click "query" to get the time spans of the plots.
The plots are sorted at time span end-date. Oldest end-dates are first. The entry at the right of each row ("x days") means the time span of day between this time span end-date and the latest end-date of this set plots.
When there are long time spans between this plot end-date and the latest plot end-date the row is colored yellow, orange or red.
